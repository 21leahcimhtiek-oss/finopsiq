# FinOpsIQ - Sales and Go-to-Market Guide

## Elevator Pitch

FinOpsIQ gives engineering teams a single pane of glass for cloud costs across AWS, GCP, and Azure. Our AI detects anomalies before they blow up your budget, ML models forecast spend 90 days out, and automated recommendations tell you exactly which resources to rightsize. Most customers save 20-35% on cloud spend within 90 days.

## Target Customers

### Primary ICP (Ideal Customer Profile)
- Company size: 50-500 employees

- MSPs and consultancies reselling cloud cost optimization to clients

## Objection Handling

**We already use the native cloud cost tools.**
Native tools do not show multi-cloud consolidated view, do not detect anomalies with AI. FinOpsIQ adds intelligence on top.

**We can build this ourselves.**
Building this takes 3-6 months of engineering time. FinOpsIQ is 99/mo and ready today.

**We are not ready to give API access to cloud accounts.**
Start with CSV upload - no API keys required. Graduate to API connectors when comfortable.
